## 🧩 Topics Covered

### 1️⃣ Arrays
- Kadane's Algorithm, Moore's Voting, Dutch National Flag
- 2-Sum, 3-Sum, 4-Sum, Merge Intervals
- Next Permutation, Stock Buy & Sell, Rotate Matrix

### 2️⃣ Linked List
- Reverse LL, Detect & Remove Cycle (Floyd's)
- Merge Sort on LL, LRU Cache
- Clone LL with Random Pointers

### 3️⃣ Greedy
- Activity Selection, Job Sequencing
- N meetings in one room, Minimum Platforms

### 4️⃣ Recursion & Backtracking
- N-Queens, Sudoku Solver, Rat in a Maze
- Subsets, Permutations, Combination Sum

### 5️⃣ Binary Search
- Search in Rotated Sorted Array
- Median of 2 Sorted Arrays, Allocate Books
- Aggressive Cows, Kth element of 2 sorted arrays

### 6️⃣ Stack & Queue
- Monotonic Stack (NGE, Previous Smaller)
- LFU / LRU Cache, Sliding Window Maximum
- Largest Rectangle in Histogram

### 7️⃣ Binary Tree
- Traversals (Inorder, Preorder, Postorder, Level Order)
- LCA, Max Path Sum, Burning Tree
- Construct Tree from Inorder + Preorder/Postorder

### 8️⃣ Graph
- BFS, DFS, Topological Sort (BFS + DFS)
- Dijkstra, Bellman-Ford, Floyd Warshall
- Kosaraju's, Bridges & Articulation Points
- DSU (Disjoint Set Union) — Union by Rank & Size

### 9️⃣ Dynamic Programming
- 1D, 2D, DP on Grids, Strings, Trees, MCM
- LCS, LIS, Edit Distance, Matrix Chain Multiplication
- Partition DP, Stocks DP, Knapsack variants

### 🔟 Tries
- Insert, Search, StartsWith
- Count Distinct Substrings, Maximum XOR
